Цель проекта заключается в анализе и выявлении ключевых факторов, влияющих на оптимальное содержание железа в рудном концентрате, получаемом на горно-обогатительных комбинатах.

В репозитории находится 6 файлов:
1) Project_Dorofeev_23_11_2025 - Jupiter Notebook 
2) SENSORS.csv - параметры датчиков на производстве - доступен по ссылке https://drive.google.com/file/d/1pEzfc1nzkXa8PtdrNALikp-6GBT51Qn-/view?usp=sharing
3) SENSORS_INFO.csv - информация о датчиках
4) RESULTS.csv - содержание
5) Презентация для защиты (недоработана)
6) PDF-документ описания проекта
